# Site Lidiane Barreto - Terapeuta

Site institucional para Lidiane Barreto, Terapeuta Familiar, de Casal e Psicóloga.

## Características

- Design moderno e responsivo
- Animações suaves
- Formulário de contato interativo
- Otimizado para dispositivos móveis
- Suporte a atendimento online e presencial

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Font Awesome (ícones)
- Google Fonts

## Estrutura do Projeto

```
SiteLidiane/
├── css/
│   └── style.css
├── js/
│   └── main.js
├── img/
│   ├── logoLidiane.jpg
│   └── fotoLidiane.jpg
├── index.html
└── README.md
```

## Como Usar

1. Clone o repositório
2. Abra o arquivo `index.html` em seu navegador
3. Para desenvolvimento, use o Live Server
